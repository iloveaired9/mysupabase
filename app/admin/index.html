⚡ 빠른 명령어

Docker

# 서비스 시작
docker-compose up -d
# 서비스 중지
docker-compose down
# 로그 확인
docker-compose logs -f

API 테스트

# 게시글 조회
curl "http://localhost:3001/posts?limit=5"
# 카테고리 필터
curl "http://localhost:3001/posts?categories.name=eq.tech"
# 통계 조회
curl -X POST http://localhost:3001/rpc/get_posts_stats -d "{}"

포트 확인

# 모든 포트 확인
netstat -an | grep LISTEN
# 특정 포트 확인
lsof -i :3001
# Docker 상태
docker-compose ps
⚠️ 중요: 이것은 개발 환경입니다. 모든 자격증명이 공개되어 있으므로 프로덕션 환경에서는 사용하지 마세요. 프로덕션에서는 환경 변수로 민감한 정보를 관리하고, 강력한 암호를 설정하세요.
